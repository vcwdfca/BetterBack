# BetterBack
The original intention was to port the right-click return to the previous screen from the GTNH version to 1.12.  
Additionally, some QoL improvements have been ported over.

# About
This project ports the right-click-to-go-back feature and other Qol improvements from GTNH's BetterQuesting fork to BetterQuesting Unofficial for 1.12.2.

### Other QoL improvements:
- View depend from [GTNewHorizons/BetterQuesting#198](https://github.com/GTNewHorizons/BetterQuesting/pull/198)
- Directional arrows for quest lines from [GTNewHorizons/BetterQuesting#205](https://github.com/GTNewHorizons/BetterQuesting/pull/205)
- Global and per-chapter completion percent from [GTNewHorizons/BetterQuesting#223](https://github.com/GTNewHorizons/BetterQuesting/pull/223)
